# QyCore

编辑器的核心业务。

## 负责的范围

提供一个通用的编辑器应用以及一系列用于后续开发的相关协议或接口，主要用于对时间序列（或时间序列相关的序列）进行处理、对实时性要求不高的场景。

其主要包括：

* 参数的管理
  * 曲线工具（贝塞尔曲线）
* 对参数的操作
